# digital-literacy-project
NGMC_project
This project is about building a Digital Literacy Portfolio as part of my role as a Student Digital Ambassador. The idea is to show that I can guide my peers in using digital tools responsibly and professionally. The portfolio is organized in a GitHub repository and covers five tasks: creating an infographic on digital literacy, setting up professional online profiles, exploring coding and collaboration platforms, drafting professional emails with etiquette guidelines, and preparing a cybercrime awareness case study with prevention tips. Along with the repository, I will submit a written project report that documents my work, reflections, and learning. The goal is to develop practical digital skills that will help me and my batchmates in academics, internships, and future careers.
